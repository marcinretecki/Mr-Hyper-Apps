# MR Hyper Apps
Custom Karabiner setup

####  Create Hyper `⚡︎` mod key (`⌘` + `⌥` + `⌃` + `⇧`)

## Apps

| `⚡︎`| Open  | Comment  |
| ------ | ---------- | ------------ |
|`⚡︎ B`  | Firefox  | `B` is for browser   |
| `⚡︎ S`  | Sublime Text |  |
| `⚡︎ G`   | Sourcetree | `G` is for git  |
| `⚡︎ T`   | Transmit |  |
| `⚡︎ E`   | Finder |  |



## Together with Spectacle
