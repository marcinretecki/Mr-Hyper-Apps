# Mr-Hyper-Apps
Custom Karabiner setup

####  Create Hyper `⚡︎` mod key (`⌘` + `⌥` + `⌃` + `⇧`)

## Apps

| `⚡︎ +`| Open  | Comment  |
| ------ | ---------- | ------------ |
|`⚡︎ B`  | Firefox  | `B` is for browser   |
| `⚡︎ S`  | Sublime Text |  |
| `⚡︎ G`   | Sourcetree | `G` is for git  |



## Together with Spectacle

| `⚡︎ +`| Window Management  | Comment  |
| ------ | ---------- | ------------ |
| `⚡︎ F`   | Full screen |   |
|`⚡︎ ←`  | Left half  |  |
| `⚡︎ →`   | Right half |  |
| `⚡︎ ↓`  | Lower half |  |
| `⚡︎ ↑`   | Upper half |   |
| `⚡︎ 3`   | Next third |   |
